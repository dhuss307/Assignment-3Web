Inventory System

Another one of my projects I have been working on is a Inventory system with Java. The libraries I used for this program are JavaFX and the utilities that are imbedded within this plugin. To design the program I used an app called Scenebuilder. This made it easier to design as it would require me to learn a completely different language. This program required me to face challenges and overcome these challenges.
